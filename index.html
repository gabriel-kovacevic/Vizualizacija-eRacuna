TEST Vizualizacija eRačuna

Kako koristiti?

Pritiskom na gumb Select one or more XML files, otvorit će se dijalog za upload XML dokumenata. Kad odaberete XML dokumente računa koje želite vizualizirati, automatski će se pojaviti vizualizacija svakog od računa. Pritiskom na gumb Print All as PDF otvorit će se print dijalog za svaki od računa.

Ako ih želite spremiti u PDF formatu, odaberite Save as PDF ili Microsoft Print to PDF kao printer. Pritiskom na gumb Export All as HTML preuzeti ćete sve vizualizirane račune u ZIP datoteci u HTML formatu.

Osim preuzimanja svih računa, možete preuzeti i pojedini vizualizirani račun pritiskom na Print as PDF gumb. Otvorit će se isti print dijalog, te je također potrebno odabrati PDF printer.
